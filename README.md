# Thiago's Vim Setup

My messy dotfiles.

## Installation:

Usable right off the bat, just clone the repo and fire setup.sh.

    $ git clone git://github.com/thiagoa/dotvim.git ~/.vim 
    $ ~/.vim/setup.sh

## Create per machine configuration (optional):

    $ touch ~/.vimrc.local
    $ touch ~/.gvimrc.local
